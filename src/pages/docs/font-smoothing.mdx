## Basic usage

### Applying font smoothing

Use the `subpixel-antialiased` utility to render text using subpixel antialiasing and the `antialiased` utility to render text using grayscale antialiasing.

<Example>
  <div class="flex flex-col gap-8">
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">subpixel-antialiased</span>
      <p class="subpixel-antialiased text-lg font-medium text-gray-900">
        The quick brown fox jumps over the lazy dog.
      </p>
    </div>
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">antialiased</span>
      <p class="antialiased text-lg font-medium text-gray-900">
        The quick brown fox jumps over the lazy dog.
      </p>
    </div>
  </div>
</Example>

```html
<p class="**subpixel-antialiased** ...">The quick brown fox ...</p>
<p class="**antialiased** ...">The quick brown fox ...</p>
```
